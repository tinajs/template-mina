> Scaffolding out a Mina project.

## Quick Overview
```bash
npm i -g sao

sao mina my-app
cd my-app
npm start
```

Then add ``dist`` directory to [Wechat Web Devtools](https://mp.weixin.qq.com/debug/wxadoc/dev/devtools/download.html) and see your app.
